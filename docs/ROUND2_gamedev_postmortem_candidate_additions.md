# Candidate additions for the Game Developer / Gamasutra postmortems list

These are the entries from the omission shortlist, formatted for manual review and import.

## Core additions

### 1. [Postmortem: The making of Valve's Half-Life 2 (2004)](https://www.gamedeveloper.com/design/classic-postmortem-the-making-of-i-half-life-2-i-)

- **Game / subject:** Half-Life 2
- **By:** Brian Jacobson, David Speyrer
- **Online publication date:** 2015-11-16
- **Original source:** Game Developer Magazine, November 2005
- **Why include:** Essential companion to The Cabal / Half-Life and a direct sequel-process article.

### 2. [Postmortem: The singular design of Namco's Katamari Damacy (2004)](https://www.gamedeveloper.com/design/postmortem-the-singular-design-of-namco-s-katamari-damacy-2004-)

- **Game / subject:** Katamari Damacy
- **By:** Keita Takahashi
- **Online publication date:** 2024-12-20
- **Original source:** Game Developer Magazine, December 2004
- **Why include:** Canonical auteur/process piece; high cultural signal.

### 3. [Classic Postmortem: Double Fine's Psychonauts](https://www.gamedeveloper.com/audio/classic-postmortem-double-fine-s-i-psychonauts-i-)

- **Game / subject:** Psychonauts
- **By:** Caroline Esmurdoc
- **Online publication date:** 2015-08-17
- **Original source:** Game Developer Magazine, August 2005
- **Why include:** Obvious Double Fine gap, especially with Brutal Legend already included.

### 4. [Baldur's Gate II: The Anatomy of a Sequel](https://www.gamedeveloper.com/design/-i-baldur-s-gate-ii-i-the-anatomy-of-a-sequel)

- **Online publication date:** 2000-10-05
- **Original source:** Game Developer / Gamasutra-era article
- **Why include:** BioWare ur-text; listed by Game Developer among seminal postmortems.

- **Online publication date:** 2015-10-30
- **Original source:** Game Developer Magazine, March 2005
- **Why include:** Horror-canon entry with notable author signal from Yamaoka.

### 6. [Classic Postmortem: Deadly Premonition](https://www.gamedeveloper.com/design/classic-postmortem-i-deadly-premonition-i-)

- **Game / subject:** Deadly Premonition
- **By:** Access Games team / Game Developer staff presentation
- **Online publication date:** 2015-08-31
- **Original source:** Game Developer Magazine, August 2010

- **Original source:** Game Developer Magazine, April 2004
- **Why include:** Postmortem under a 'making of' title; easy to miss, but very much in-scope.

### 8. [Classic Postmortem: Obsidian's Knights of the Old Republic II](https://www.gamedeveloper.com/design/classic-postmortem-obsidian-s-i-knights-of-the-old-republic-ii-the-sith-lords-i-)

- **Original source:** Game Developer Magazine, April 2005
